## 一个完整的 Go 语言博客系统后端 API。整个项目：

✅ 已完成的核心功能

📦 项目结构
- cmd/main.go - 程序入口，初始化数据库和路由
- config/database.go - 数据库连接和自动迁移
- models/ - 三个数据模型（User, Post, Comment）
- controllers/ - 三个控制器（认证、文章、评论）
- middleware/ - JWT 认证和日志中间件
- utils/ - JWT 工具和统一响应格式
- routes/routes.go - 完整的路由配置

- ✅ 用户注册（密码 bcrypt 加密）

- ✅ 获取用户信息（需要认证）
- ✅ JWT 中间件保护路由

💬 评论系统

## 📝 使用说明

1. 启动后端服务
2. cd golang_blog
go run cmd/main.go

### 2. 打开前端界面
直接在浏览器中打开生成的 HTML 文件

### 3. 配置 API 地址
如果后端不在 `localhost:8080`，修改页面顶部的 API 地址

### 4. 测试流程

#### 第一步：注册/登录
1. 切换到「认证测试」标签
2. 填写注册信息（用户名至少3个字符，密码至少6位）
3. 点击注册，自动获取 token
4. 或使用已有账号登录

#### 第二步：创建文章
1. 登录后，切换到「文章管理」标签
2. 点击「创建文章」按钮
3. 填写标题和内容
4. 保存文章

#### 第三步：管理文章
1. 在「文章列表」查看所有文章
2. 点击「查看详情」阅读完整内容
3. 在「我的文章」中编辑或删除自己的文章

#### 第四步：发表评论
1. 点击任意文章的「查看详情」
2. 在底部评论框输入内容
3. 点击「发表评论」

#### 第五步：查看统计
1. 切换到「系统统计」标签
2. 查看各项统计数据
3. 点击刷新按钮更新数据
